# Job Application Database

Welcome to the Job Application Database project! This repository contains the SQL code for creating and managing a database designed to help students track their job applications throughout the job-seeking process. This database serves as a structured way to store information about applications, including company details, positions, submission dates, and application statuses.

## 🌟 Features

- **Structured Database**: A well-organized database schema to store job application data efficiently.
- **Application Tracking**: Capabilities to manage multiple job applications, helping students stay organized during their job search.
- **Flexibility**: Easily extendable to accommodate additional features, such as tracking interviews or notes.

## 🛠️ Technologies Used

This project utilizes the following technologies:

- **SQL**: For database management and query operations.
- **SQL Developer**: The tool used for database development and management.
